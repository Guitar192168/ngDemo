## 服务  依赖注入

    ng g service  my-new-service
    创建到指定目录下
    demo:  ng g service services/request
    这里会新增一个services文件夹，创建一个request的服务。全局可以
    使用

    创建完需要在全局导入
    还需要在要使用的页面进行导入

    import { RequestService } from 'src/app/services/request.service'; 

    constructor(public request: RequestService) { }
    
  * import { RequestService } from 'src/app/services/request.service'; *
